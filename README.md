# คู่มือการใช้งานเว็บแอปพลิเคชัน "บันทึกข้อมูลส่วนตัว (Note App)"

แอปพลิเคชันนี้ทำงานบนเว็บไซต์ และเชื่อมต่อกับ Google Sheets เพื่อบันทึกข้อมูลแบบเรียลไทม์

## ขั้นตอนที่ 1: เตรียม Google Sheets และ โค้ด Backend
1. สร้าง **Google Sheet** ใหม่ขึ้นมา (ไม่ต้องใส่ข้อมูลอะไรเข้าไป ระบบจะสร้างให้เอง)
2. ไปที่เมนู **ส่วนขยาย (Extensions)** > **Apps Script**
3. ลบโค้ดเก่าทั้งหมดในหน้าต่างนั้นออกให้หมด
4. ไปที่ไฟล์ `Code.gs` ที่อยู่ในโฟลเดอร์นี้ **คัดลอกโค้ดทั้งหมด** แล้วนำไปวางในหน้าต่าง Apps Script
5. กดปุ่ม `Ctrl + S` (หรือ Cmd + S บน Mac) เพื่อทำการบันทึก
6. *เลือกเมนูด้านบน เลือกรันฟังก์ชั่น `setupSheets` แล้วกดปุ่ม "เรียกใช้" ระบบจะขอสิทธิ์เข้าถึงบัญชี ให้กดยืนยันให้สิทธิ์ทั้งหมด (ขั้นตอนนี้อาจต้องกด 'ขั้นสูง' > 'ไปยังโค้ดที่ไม่ปลอดภัย')*

## ขั้นตอนที่ 2: แปลงเป็น Web App (API)
เมื่อนำโค้ดไปวางเสร็จแล้ว ต้องทำการทำให้มันกลายเป็น Web API
1. กดที่ปุ่ม **การทำให้ใช้งานได้ (Deploy)** ที่มุมขวาบน > เลือก **การทำให้ใช้งานได้รายการใหม่ (New deployment)**
2. คลิกไอคอนรูปเฟือง (⚙️) ข้างๆ หัวข้อ "เลือกประเภท" แล้วเลือก **เว็บแอป (Web App)**
3. ในช่อง "คำอธิบาย": พิมพ์อธิบายสั้นๆ (เช่น API 1.0)
4. ในช่อง "สิทธิ์เข้าถึง (Who has access)": ให้เลือก **"ทุกคน (Anyone)"** <--- *(สำคัญมาก หากไม่เลือกตัวนี้ ตัวแอปหน้าเว็บจะบันทึกไม่ได้)*
5. กดปุ่ม **การทำให้ใช้งานได้ (Deploy)** แถบสีน้ำเงินด้านล่างสุด
6. คัดลอกลิงก์ **URL ของเว็บแอป** ที่ได้เก็บไว้

## ขั้นตอนที่ 3: เปิดใช้งานแอปพลิเคชัน (Frontend)
1. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
2. ในแอปพลิเคชัน ให้กดปุ่ม **รูปเฟืองการตั้งค่า** มุมขวาบน
3. ไปที่แท็บ **"ส่วนเชื่อมต่อ Web App"**
4. นำ URL ของเว็บแอป (จากขั้นตอนที่ 2 ข้อที่ 6) มาวางในช่อง แล้วกดปุ่ม **"บันทึก URL & โหลดข้อมูล"**
5. ตอนนี้แอปพร้อมใช้งานแล้ว! ลองกดบันทึกข้อมูลดูได้เลย ระบบจะส่งข้อมูลเข้าไปอยู่ใน Google Sheets ของคุณทันทีในชีตชื่อ "Notes"

## การปรับแต่งหมวดหมู่
- ในแอปคลิกปุ่ม **ตั้งค่า (รูปเฟือง)** > **จัดการหัวข้อ**
- คุณสามารถเพิ่มไอคอน (ใช้คีย์บอร์ด Emoji หรือก๊อปปี้ Emoji มาวาง) และสามารถแก้ไขรายชื่อวิชา / โหมดหมู่ย่อยได้ โดยให้ใช้ลูกน้ำ (`,`) คั่นระหว่างรายชื่อ
- พอกดบันทึก มันจะบันทึกลงในชีต "Categories" ใน Google Sheet ของคุณด้วย
- ตัวเลือกไหนที่ไม่มี หรืออยากแทรกเฉพาะครั้งนั้นๆ สามารถปล่อยให้ "หัวข้อย่อย" เป็นตัวเลือก "ระบุเอง" แล้วพิมพ์สิ่งที่ต้องการลงไปได้เลย

## การนำเว็บขึ้นออนไลน์ด้วย Vercel (เว็บโหลดเร็ว และรองรับ PWA)
ระบบแอปนี้รองรับเทคโนโลยี PWA ทำให้สามารถติดตั้งลงมือถือได้เลยเหมือนแอปปกติ
1. **สมัครสมาชิก Vercel:** ไปที่ [vercel.com](https://vercel.com/) (แนะนำให้ล็อกอินด้วย GitHub)
2. **สร้างโปรเจกต์ใหม่:** 
   - วิธีที่ง่ายที่สุดคือการนำไฟล์ทั้งหมดในโฟลเดอร์นี้อัปโหลดขึ้น GitHub Repository ของคุณ
   - จากนั้นไปที่ Vercel กด **Add New Project** แล้ว Import เว็บของคุณเข้ามา
   - กด **Deploy** แล้วรอสักครู่ ระบบจะสร้างลิงก์สำหรับเข้าใช้งานให้ (ลงท้ายด้วย `.vercel.app`)
3. **ติดตั้งเป็นแอปลงมือถือ (PWA):**
   - นำลิงก์ที่ได้จาก Vercel ไปเปิดใน Google Chrome (สำหรับ Android) หรือ Safari (สำหรับ iOS) 
   - กดที่เมนู หรือปุ่มแชร์ แล้วเลือก **"Add to Home Screen" (เพิ่มลงในหน้าจอโฮม)**
   - ตอนนี้คุณสามารถเปิดแอปบันทึกข้อมูลได้จากหน้าจอมือถือโดยตรง!
